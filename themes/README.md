## Available Themes

- **aetheria** - Purple/blue ethereal theme
- **ayu-mirage** - Warm desert tones
- **catppuccin** - Mocha pastel palette
- **catppuccin-latte** - Light mode variant
- **matte-black** - Minimalist black
- **nord** - Arctic cool tones
- **osaka-jade** - Custom jade/teal
- **ristretto** - Espresso browns
- **rose-pine** - Soft rose palette
- **rose-pine-dark** - Dark variant
- **tokyo-night** - Deep blues with neon
- **torrentz-hydra** - Bold contrast

## Quick Usage

```bash
fedpunk-theme-list              # List all themes
fedpunk-theme-set tokyo-night   # Set theme
fedpunk-theme-next              # Next theme (Super+Shift+T)
```
