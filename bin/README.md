## Quick Reference

### Theme Management
```bash
fedpunk-theme-list              # List available themes
fedpunk-theme-set <name>        # Set specific theme
fedpunk-theme-next              # Next theme
fedpunk-theme-prev              # Previous theme
fedpunk-theme-current           # Show current theme
```

### Wallpaper Management
```bash
fedpunk-wallpaper-next          # Next wallpaper
fedpunk-wallpaper-set           # Set wallpaper from theme
```

### Custom Dotfiles
```bash
fedpunk-stow-custom --list      # List packages
fedpunk-stow-custom <pkg>       # Deploy package
fedpunk-stow-custom --delete <pkg>  # Remove package
```
